Acerca de DigimonAPI

Descripción

DigimonAPI es una API que proporciona datos detallados sobre Digimon, permitiendo búsquedas avanzadas, filtrado y gestión de favoritos.

Tecnologías

Angular, SCSS, CSS Grid y Flex, entre otras tecnologías modernas.

Funciones

Búsqueda por nombre, filtrado por atributo, paginación, modo oscuro y animaciones interactivas.

Estilos

Uso de animaciones, transiciones, pseudoelementos, combinadores, selectores de atributos, transformaciones, mix-blend-mode, máscaras, y ahora además Grid para posicionar elementos de forma responsive.
